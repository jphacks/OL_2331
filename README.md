## 製品概要
### 背景(製品開発のきっかけ、課題等）待ち合わせの集合時間に間に合うために何時に起きたらよいかわからず、予定よりかなり早く到着してしまったり逆に見通しが甘く、5分10分遅刻したりすることがあります。
### 製品説明（具体的な製品の説明）到着の場所と時間に合わせて起きる時間や移動手段を調べてくれます。
